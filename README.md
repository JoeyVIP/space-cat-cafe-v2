# 🐱☕✨ 太空貓咖啡館 Space Cat Cafe

在浩瀚宇宙中，與貓咪共度美好時光

## 專案簡介

太空貓咖啡館是一個充滿創意的咖啡廳主題網站，結合了太空科幻元素與可愛貓咪，打造獨特的視覺體驗。

### 特色

- 🌌 深藍太空主題設計
- 🐾 可愛的太空貓咪元素
- 📱 完全響應式設計
- ✨ 流暢的動畫效果
- 🎨 精美的視覺呈現

## 技術棧

- HTML5
- CSS3 (包含動畫與漸層)
- JavaScript (原生)
- 響應式設計

## 網站結構

- **首頁** - 主視覺與品牌介紹
- **關於我們** - 咖啡館的故事與特色
- **菜單** - 完整的餐點與飲品選項
- **營業資訊** - 營業時間、地點、聯絡方式

## 部署

本專案使用 Railway 進行部署

### 本地開發

```bash
# 安裝依賴
npm install

# 啟動本地伺服器
npm start
```
